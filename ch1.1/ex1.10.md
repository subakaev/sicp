# Упражнение 1.10.
Следующая процедура вычисляет математическую функцию, называемую функцией Аккермана.

```scheme
(define (A x y)
  (cond ((= y 0) 0)
        ((= x 0) (* 2 y))
        ((= y 1) 2)
        (else (A (- x 1)
                (A x (- y 1))))))
```

Каковы значения следующих выражений?

## Expression 1
```scheme
(A 1 10)
```

```scheme
(A 1 10)
(A 0 (A 1 9))
(A 0 (A 0 (A 1 8)))
(A 0 (A 0 (A 0 (A 1 7))))
...
**2^10**
```


## Expression 2
```scheme
(A 2 4)
```
```scheme
(A 3 3)
```
